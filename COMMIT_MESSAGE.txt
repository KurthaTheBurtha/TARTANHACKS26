feat: Complete backend implementation for CareMap MVP

Implement full backend infrastructure for healthcare document analysis,
insurance-aware RAG chat, and provider search with in-network scoring.

## Core Features

### Pillar A: Document Lifecycle
- Document upload with Supabase Storage signed URLs
- Document analysis with extraction persistence
- PHI-safe logging and structured data storage
- Support for bills, EOBs, policies, and SOBs

### Pillar B: Insurance-Aware RAG Chat
- Policy document ingestion (PDF extraction, chunking, embedding)
- Vector search with pgvector for policy chunks
- Grounded chat responses with citations
- Streaming chat endpoint (SSE) with token-by-token responses
- LLM integration (OpenAI) with deterministic fallback

### Pillar C: Provider Search & In-Network Scoring
- Cache-first provider search with MockProviderSource adapter
- Policy-aware in-network scoring using RAG retrieval
- Weighted heuristics for network status evaluation
- Provider normalization and distance calculation

## Infrastructure

- FastAPI with versioned /v1 routes
- Supabase integration (Postgres, Auth, Storage, Vector)
- JWT authentication with test override mode
- Row-Level Security (RLS) with comprehensive isolation tests
- Contract tests ensuring API stability
- CI/CD with GitHub Actions
